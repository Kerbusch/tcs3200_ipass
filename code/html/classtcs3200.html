Detailed Description

TCS3200 sensor.

This class is made for the TCS3200 color sensor. The TCS3200 outputs a square wave with differing frequency.


Working of the TCS3200


The TCS3200 has a current to frequency converter build in.
This class uses that frequency to convert it back to the color value.
